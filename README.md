##  Objectives

Zest is a high-performance online assessment platform engineered for the modern web. Our goal is to eliminate the technical friction and visual clutter of traditional testing software. We provide a focused, frictionless environment that ensures:

Clarity: Minimalist design to reduce cognitive load during exams.

Speed: Rapid delivery and instantaneous scoring.

Scalability: Architecture built to handle growing user bases with ease.

##  Core Features

Fluid UI/UX: A sleek, mobile-responsive interface for seamless testing on any device.

Smart Proctoring: Lightweight monitoring tools to ensure integrity without intrusiveness.

Real-time Analytics: Instant performance insights and data visualization for both students and admins.

Dynamic Content: Support for various question types with a powerful, markdown-ready editor.

## Tech Stack (MERN)

Zest is built on the MERN stack for high performance and full-stack JavaScript consistency:

### MongoDB: 
A flexible NoSQL database used to store complex question structures and user results with high scalability.

### Express.js: 
A minimalist web framework for Node.js that handles our robust RESTful API and routing logic.

### React: 
A declarative frontend library providing a fast, component-based user interface for a smooth single-page application (SPA) experience.

### Node.js: 
The asynchronous event-driven JavaScript runtime that powers our backend, ensuring fast server-side execution.

## Getting Started

#### Note: 
Due to the proprietary nature of this project, environment variables and API keys are restricted.

## Clone the repository

Install dependencies: ```npm install``` (in root, frontend, and backend)

Configure Environment: Create a ```.env``` file based on ```.env.example```

Run development: ```npm run dev```
